    GetAntiForgeryTokenAsync(HttpResponseMessage)

    Extracts anti-forgery token from the HTTP response. Z HTTP odpovede získa hodnotu anti-forgery tokenu. Ten sa získa iba v prípade, že odpoveď je úspešná, tzn. jej StatusCode má hodnotu 2xx.

    Declaration

    Parameters
    Type Name Description
    System.Net.Http.HttpResponseMessage response

    HTTP response from which the anti-forgery token is extracted.
    Returns
    Type Description
    System.Threading.Tasks.Task<System.String>

    Value of the anti-forgery token or null, if the token was not found or the response is not successful.
    Remarks

    The token is extracted only if the respons was successful (System.Net.Http.HttpResponseMessage.IsSuccessStatusCode). The response's content is searched for form's field with the name AntiForgeryTokenFieldName. If that field is found, it's value is returned.
